
# Restaurant Website (React)

<img width="1904" height="996" alt="restaurant-site" src="https://github.com/user-attachments/assets/81a54cd2-c146-4de7-9cff-acd8e490bf2a" />

A responsive restaurant website built with React that showcases menu items and provides a simple food ordering interface.

Live demo: [https://abhinavpbabu.github.io/restaurant-site-demo/](https://abhinavpbabu.github.io/restaurant-site-demo/)

## Features

- Interactive restaurant menu
- Clean and responsive layout
- Component-based React architecture
- Simple food ordering interface
- Easily customizable for different restaurants

## Tech Stack

- React
- JavaScript
- HTML5
- CSS

## Installation

Clone the repository

```bash
git clone https://github.com/abhinavpbabu/restaurant-site.git
cd restaurant-site
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
